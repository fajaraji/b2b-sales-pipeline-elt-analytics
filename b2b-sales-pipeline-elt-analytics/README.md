# B2B Sales Pipeline Analytics

This repository contains the analytics pipeline for B2B sales data.

## Structure

- `config/` - configuration files for environment and Bruin
- `data/raw/` - raw local test data
- `docs/` - architecture and documentation
- `etl/` - extract, transform, load folders
- `pipelines/` - orchestration definitions and Bruin assets
- `src/` - shared Python utilities
- `tests/` - test files and fixtures
- `terraform/` - infrastructure as code for cloud resources
